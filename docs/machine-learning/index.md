# 机器学习

## 一些说明

Python 是我自学的第一门编程语言，近几年 Python 也确实挺火的，语法清晰、应用广泛，确实适合作为大众学习编程的入门首选。这里的文档是我决定在 Python 语言上主攻机器学习后，系统自学机器学习的笔记。适合用于自学和语法查阅。

我写这份文档时的软件版本如下：

- Python 3.10.4
- NumPy 1.22.3
- Pandas 1.4.2
- matplotlib 3.5.2
- seaborn 0.11.2
