# 数据分析

## 一些说明

Python 是快速实现数据分析的利器，而数据分析是机器学习及相关领域的基础。这份文档是我系统学习数据分析的笔记，主要围绕 NumPy、Pandas、Matplotlib、Seaborn 展开，中间也会涉及一些数学知识，内容都比较基础，适合自学时查阅。

我写这份文档时的软件版本如下：

- Python 3.10.4
- NumPy 1.22.3
- Pandas 1.4.2
- matplotlib 3.5.2
- seaborn 0.11.2
